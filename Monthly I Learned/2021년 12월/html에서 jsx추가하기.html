선택사항 JSX로 React 해보기

    가장 빠르게 JSX 시도하기
  1. 스크립트 태그를 집어넣기 src="https://unpkg.com/babel-standalone@6/babel.min.js"
  2. <.script>태그에 type="text/babel" 성질을 추가하기
    프로젝트에 npm 방식으로 JSX 추가하기
  1. npm init -y 를 실행하세요.
  2. npm install babel-cli@6 babel-preset-react-app@3를 실행하세요.

Jsx를 사용하지 않을 때의 코드

const e = React.createElement;

    // "좋아요" button을 표시
    return e(
      'button',
      { onClick: () => this.setState({ liked: true }) },
      'Like'
    );

Jsx를 사용할 때의 코드

//  "좋아요" 을 표시
    return (
      <.button onClick={() => this.setState({ liked: true })}>
        Like
      
    );
